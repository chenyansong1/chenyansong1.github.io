create [-s] [-e] path data acl

  1. 创建zookeeper节点
  2. -s或者-e表示创建的是顺序或临时节点，不加默认创建的是持久节点
  3. Path为节点的全路径，没有相对节点的表示方式
  4. data位当前创建节点内存储的数据
  5. acl 用来进行权限控制，缺省情况下不做任何权限控制
