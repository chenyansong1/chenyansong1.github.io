5.守护线程的作用：setDaemon(True)

设置守护线程的作用：当主线程退出的时候，会将子线程也结束，不管此时子线程是否执行完成，这就是守护线程的作用

6.join()

将join注释掉之后，主线程将不会等待子线程结束之后，再结束，而是直接退出，那么在主线程退出之后，子线程由于守护线程的存在也会退出

如果没有守护线程，那么子线程将不会退出（在主线程退出时）
